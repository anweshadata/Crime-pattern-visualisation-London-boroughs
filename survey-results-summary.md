- **Total responses:** 27
- **Average completion time:** 21:21
- **Collection window:** 107 days

This summary reports aggregated statistics only (response distributions and common themes). Individual free-text comments are not reproduced here, in line with the anonymity/voluntary terms given to participants.

## Common themes from open-text feedback (word-cloud frequency)

Rather than reproducing individual comments, the most frequent themes/keywords participants used per section are summarised below (based on Microsoft Forms' automatic word-cloud analysis):

| Section | Top recurring themes |
|---|---|
| Navigation demo | navigation clarity, intuitive buttons, hotspot/overview references |
| Main page | consistent button placement across pages, simple/short layout, colour scheme |
| Overview page | visuals could be improved (e.g. KPIs vs. pie chart), borough filter, generally "good" |
| Crime categories | space used by most/least common displays, generally rated "good" |
| Hotspots | request for more visible hotspot points, map clarity praised |
| Trends | x-axis labelling (months vs. years) flagged as confusing, otherwise well received |
| Youth-related crime | comparison framing (youth vs. general crime) needs clarifying, page otherwise clear |
| Summary | suggestion to convert text into bullet points/shorter blocks for readability |
| Overall | requests for larger/consistent navigation buttons, thematic colour consistency |

---

## Notes on methodology

- Percentages are based on the number of respondents who answered each question (24–27 per question; a small number skipped optional follow-ups).
- Word-cloud themes reflect Microsoft Forms' automatic keyword extraction from free-text answers, not verbatim quotes.
- Raw individual responses are retained privately by the researcher and are not published here to protect participant confidentiality.
